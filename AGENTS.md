# AGENTS.md

## Project Overview

This is an Obsidian vault for personal notes and knowledge management. It contains Markdown files with wiki-style linking.

## Communication Language

- All communication with the user MUST be in Russian language
- All thinking blocks and internal reasoning MUST be written in Russian
- File content can be in Russian or English depending on context

## Build/Lint/Test Commands

This is a notes repository - no build, lint, or test commands are applicable.

## Code Style Guidelines

### Markdown Files

- Use UTF-8 encoding for all files
- Prefer Russian or English language based on existing content
- Use wiki-style links: `[[filename]]` or `[[filename|display text]]`
- Use standard Markdown formatting:
  - Headers: `#`, `##`, `###`
  - Bold: `**text**`
  - Italic: `*text*`
  - Lists: `-` or `1.`
  - Code: `` `inline` `` or fenced blocks with ` ``` `

### File Naming

- Use descriptive names in the appropriate language
- Avoid special characters in filenames
- Use spaces or hyphens for readability

### Organization

- Group related notes in subdirectories when needed
- Use tags within files: `#tag-name`
- Link related notes together using wiki links

### Metadata (Optional)

YAML frontmatter can be used at the top of files:

```yaml
---
title: Note Title
date: 2024-01-15
tags: [tag1, tag2]
---
```
